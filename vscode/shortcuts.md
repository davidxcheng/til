# VS Code Shortcuts: The Short List

| Keyboard shortcut | Description |
|:---|:---|
| `⌘` + `K`, `⌘` + `S` | Show keyboard shortcuts and also search for shortcuts. Click the :keyboard: icon next to the search box to search by pressing the shortcut. |

## Windows, Panels and Tabs

`⌘` + `⇧` + `P` + `Developer: Reload Window`

| Keyboard shortcut | Description |
|:---|:---|
| `⌘` + `+` / `-` | Zoom in/out (increase/decrease font-size) |
| `⌘` + `B` | Toggle the side bar |
| `⌘` + `J` | Toggle the bottom panel |
| `⌘` + `⌃` + `⏵` / `⏴` | Split and move tab right/left |

## Editing

| Keyboard shortcut | Description |
|:---|:---|
| `⌘` + `D` | Select current word, repeat to also select next occurrence |
| `⌘` + `⇧` + `L` | Select all occurrences in file |
| `⌥` + `⇧` + `⏷` | Copy line down |
| `⌥` + `⇧` + `I` | Cursor at the end of each selected line |
| `⌘` + `⇧` + `K` | Delete line (without copying it) |

## Reading Code

| Keyboard shortcut | Description |
|:---|:---|
| `⌘` + `⇧` + `˽` | Trigger parameter hints when the cursor is inside a method call |
| `⌃` + `-` | Go back to the previous position of the cursor |
| `F12` | Go to definition |
| `⌘` + `F12` | Go to implementations |
| `⇧` + `F12` | Shows all references as a view-in-view panel. `Esc` to close |
| `⌥` + `⇧` + `F12` | Shows all references in the side bar |
