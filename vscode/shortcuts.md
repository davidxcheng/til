# VS Code Shortcuts: The Short List

| Keyboard shortcut | Description |
|:---|:---|
| `Cmd` + `K`, `Cmd` + `S` | Show keyboard shortcuts and also search for shortcuts. Click the :keyboard: icon next to the search box to search by pressing the shortcut. |

## Windows, Panels and Tabs

| Keyboard shortcut | Description |
|:---|:---|
| `Cmd` + `+` / `-` | Zoom in/out (increase/decrease font-size) |
| `Cmd` + `B` | Toggle the side bar |
| `Cmd` + `J` | Toggle the bottom panel |
| `Cmd` + `Ctrl` + :arrow_right: / :arrow_left: | Split and move tab right/left |

## Editing

| Keyboard shortcut | Description |
|:---|:---|
| `Cmd` + `D` | Select current word, repeat to also select next occurrence |
| `Cmd` + `Shift` + `L` | Select all occurrences in file |
| `Option` + `Shift` + :arrow_down: | Copy line down |
| `Cmd` + `Shift` + `K` | Delete line (without copying it) |

## Reading Code

| Keyboard shortcut | Description |
|:---|:---|
| `Cmd` + `Shift` + `Space` | Trigger parameter hints when the cursor is inside a method call |
| `Ctrl` + `-` | Go back to the previous position of the cursor |
| `F12` | Go to definition |
| `Cmd` + `F12` | Go to implementations |
| `Shift` + `F12` | Shows all references as a view-in-view panel. `Esc` to close |
| `Option` + `Shift` + `F12` | Shows all references in the side bar |
